# 错误码设计

> **文档版本**: 1.0.0  
> **更新日期**: 2026-07-09  
> **适用范围**: NPS 问卷投放模块  
> **关联文档**: `requirement/modules/05.01~05.05`

---

## 1. 错误码分段

| 错误码段 | 范围 | 类别 | 说明 |
|----------|------|------|------|
| 0 | 0 | 成功 | success |
| 1xxx | 1000-1999 | 通用/参数 | 参数校验、限流、签名等 |
| 2xxx | 2000-2999 | 认证/权限/用户 | 登录、Token、角色、用户 |
| 41xx | 4100-4199 | NPS 业务 | 投放计划、投放数据、问卷相关业务异常 |
| 9xxx | 9000-9999 | 系统/外部 | 数据库、缓存、第三方服务 |

---

## 2. 业务错误码（41xx）

| 错误码 | 错误信息 | 触发场景 | HTTP 状态码 | 处理建议 |
|--------|----------|----------|-------------|----------|
| 4101 | URL allocation percentages must sum to 100 | 保存计划时 URL 比例之和不等于 100% | 400 | 检查并修正 URL 比例 |
| 4102 | Plan code already exists | 新建/透传时 `campaign_code` 已存在 | 409 | 更换计划编码或更新已有计划 |
| 4103 | Import file format error | 投放数据导入文件格式错误 | 400 | 下载模板并按模板填写 |
| 4104 | Survey already completed | 完成接口被重复调用 | 200（幂等）/ 409 | 无需处理，已是完成状态 |
| 4105 | Delivery record not found | 完成接口或推送时投放记录不存在 | 404 | 检查 campaignCode + oneId 是否匹配投放记录（或检查 deliveryId） |
| 4106 | Plan not found | 操作计划时记录不存在 | 404 | 检查 planId / campaign_code |
| 4107 | Push not allowed | 尝试推送已推送记录 | 409 | 仅推送未推送或推送失败记录 |
| 4108 | Plan disabled or not in dispatch window | 计划停用或不在起止时间内 | 400 | 启用计划或调整时间 |
| 4109 | URL code not found in plan | 分配 URL 时找不到对应 url_code | 500 | 检查计划 URL 配置 |
| 4110 | Dispatch condition not matched | 用户不满足下发条件 | 200（跳过） | 无需处理 |

---

## 3. 通用错误码（1xxx）

| 错误码 | 错误信息 | 触发场景 | HTTP 状态码 |
|--------|----------|----------|-------------|
| 1001 | Parameter validation error | 请求参数校验失败 | 400 |
| 1002 | Request parameter missing | 缺少必填参数 | 400 |
| 1003 | Invalid parameter format | 参数格式错误 | 400 |
| 1004 | Request too frequent | 请求过于频繁，触发限流 | 429 |
| 1005 | Invalid request signature | 签名错误 | 400 |

---

## 4. 认证/权限错误码（2xxx）

| 错误码 | 错误信息 | 触发场景 | HTTP 状态码 |
|--------|----------|----------|-------------|
| 2001 | Unauthorized | 未登录或 Token 无效 | 401 |
| 2002 | Token expired | Token 已过期 | 401 |
| 2003 | Permission denied | 无权限访问 | 403 |
| 2004 | Invalid internal key | 服务间调用密钥错误 | 403 |

---

## 5. 系统/外部错误码（9xxx）

| 错误码 | 错误信息 | 触发场景 | HTTP 状态码 |
|--------|----------|----------|-------------|
| 9001 | Database error | 数据库异常 | 500 |
| 9002 | Redis error | 缓存异常 | 500 |
| 9003 | External service timeout | 外部服务调用超时 | 504 |
| 9004 | External service error | 外部服务返回错误 | 502 |
| 9999 | System internal error | 未知系统异常 | 500 |

## 7. 错误码与 BR/EX 映射

| BR/EX 编号 | 错误码 | 说明 |
|-----------|--------|------|
| BR-05.01-02 / EX-05.01-01 | 4101 | URL 比例之和不等于 100% |
| BR-05.01-01 / EX-05.02-02 | 4102 | 计划编码已存在 |
| EX-05.02-01 | 4103 | 导入文件格式错误 |
| EX-05.05-04 | 4104 | 问卷已完成 |
| EX-05.05-03 | 4105 | 投放记录不存在 |
| EX-05.01-05 等 | 4106 | 计划不存在 |
| EX-05.02-03 | 4107 | 不允许重复推送 |
| EX-05.03-02 | 9003 | CSI 推送超时 |
