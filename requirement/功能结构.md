# 功能结构

## 1. 产品功能总览

```
NPS 问卷投放
├── 1.1 NPS 问卷投放管理
│   ├── 05.01 投放计划管理
│   │   ├── 投放计划列表
│   │   ├── 新建投放计划
│   │   ├── 编辑投放计划
│   │   ├── 查看投放计划
│   │   ├── 启用/停用投放计划
│   │   └── 删除投放计划
│   ├── 05.02 投放数据管理
│   │   ├── 投放数据列表
│   │   ├── 导入投放数据
│   │   ├── 导出投放数据
│   │   ├── 推送投放数据至 CSI
│   │   └── 删除投放数据
│   ├── 05.03 定时下发任务
│   │   ├── 按国家/车型/绑车时间筛选用户
│   │   ├── 按比例分配问卷链接
│   │   ├── 生成用户投放记录
│   │   └── 回流传投数据至 CSI
│   ├── 05.04 App 问卷入口
│   │   ├── 问卷数据查询
│   │   ├── 显示规则判断
│   │   ├── 浮标展示与关闭
│   │   └── 本地 coolOff 存储
│   └── 05.05 问卷页
│       ├── WebView 加载 CSI H5
│       ├── JSBridge 完成回调
│       └── 完成状态上报
```

## 2. 模块依赖关系

| 模块 | 依赖上游 | 被依赖下游 |
|------|----------|-----------|
| 1.1 NPS 问卷投放管理 | 用户服务（oneId/车型）、车辆服务（绑车时间）、CSI 系统（计划来源）、UOP（透传） | 数据分析/报表（可选） |

## 3. 功能优先级

| 模块 | 子功能 | 优先级 |
|------|--------|--------|
| 1.1 | 投放计划管理 | P0 |
| 1.1 | 定时下发任务 | P0 |
| 1.1 | App 问卷入口 | P0 |
| 1.1 | 问卷页（H5 容器） | P0 |
| 1.1 | 投放数据管理 | P0 |
| 1.1 | 投放数据导入/导出 | P1 |

## 4. 关键术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 投放计划 | Campaign Plan | 由 CSI 或管理后台创建的 NPS 问卷投放配置 |
| 投放数据 | Delivery | 实际下发到用户的问卷记录 |
| 投放位置 | Position | 客户端展示问卷入口的位置，如车控页右下角 |
| 冷却时长 | Cool Off | 用户关闭入口后再次显示需要等待的时间 |
| 显示时长 | Display Duration | 计划配置的天数；服务端据此计算 start_time/end_time 后下发给 App 端 |
